# Single HTML Workout

单文件，HTML，健身打卡app。

## 亮点

- 自定义训练计划
- 每日训练记录
- 热力图统计
- 历史回放
- JSON 导入导出
- 无后端，无需账号

## 快速开始

直接用浏览器打开 `index.html`，或部署到 GitHub Pages。

在线测试：[点击这里](http://htmlpreview.github.io/?https://github.com/Caturra000/single-html-workout/blob/master/index.html)。

## 存储方式

数据通过 IndexedDB 保存在浏览器本地。
