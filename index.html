☁️ GitHub 云同步

通过 GitHub 仓库实现跨设备数据同步。需要创建一个 Fine-grained PAT。

1
新建一个私有仓库（如 my-workout-sync），专门存放同步数据。
2
打开 Token 设置页 ↗，点 Generate new token。
3
Token name 随意填，Expiration 建议选 90 天或 1 年。
4
Repository access 选 Only select repositories，仅勾选你的同步仓库。
5
点 Add permissions，勾选 Contents 并设为 Read and write，其余不动。
6
点 Generate token，立刻复制 github_pat_xxx…（只显示一次）。
7
在下方填入 Token 和仓库（格式 用户名/仓库名），💾 保存配置 即可。
8
为了避免后续错误操作，可以考虑先本地 📥 导出完整数据，随时都可以通过 📤 导入恢复数据 挽救回来。
9
如果你的本地设备已有数据，但未曾同步过，选择 ⬆️ 推送 到 GitHub；如果你使用新的本地设备，需要同步已有的 GitHub 远程数据，选择 ⬇️ 拉取 到本地；如果两端都没有数据？那你随便。
